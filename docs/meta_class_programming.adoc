== Метапрограммирование классов

NOTE: Метапрограммирование классов -- это искусство создания или настройки классов во время выполнения.

WARNING: Необходимо твёрдо запомнить, что все классы являются экземплярами `type`, а метаклассы -- еще и подклассами `type`, поэтому они работают как фабрики классов. В частности, метакласс может настраивать экземпляры посредством реализации `\\__init__`. Метод `\\__init__` метакласса может делать всё, на что способен декоратор, но его действие распространяется глубже.
